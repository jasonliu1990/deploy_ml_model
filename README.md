# 使用flask佈署機器學習模型



## folder
  * `api`: 使用`flask`製作的RESTful api
  * `model`: 存放訓練好的model`.pkl`檔
  * `webapp`: 使用flask製作的webapp
## jupyter notebook
  * `iris_create_model.ipynb`: 訓練模型, 存成`.pkl`檔
  * `iris_gui.ipynb`: Demo用, 使用ipywidgets
## webapp/api 使用方法
  * 用`cmd`輸入 `python 檔案.py`掛起服務, 瀏覽器開啟 `http://127.0.0.1:5000/`
